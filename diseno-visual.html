Diseño Visual

El Diseño Visual se centra en la estética y el aspecto final de un producto digital. Utiliza elementos como la tipografía, el color y el *layout* para mejorar la usabilidad, guiar al usuario y crear una identidad de marca coherente.

2. Elementos Clave

  • **Tipografía:** La elección de fuentes, tamaño y espaciado (legibilidad y jerarquía).
